Description

This is a combined ruler and protractor app for Firefox OS.

The ruler uses both cm and inch, tap the label to switch sides. The protractor meassures the angle of the long edge of your device (at least it's the long edge for a smartphone) against the place where it was when you resetted the protractor.

Note that the protractor relies on accurate data from the compass, which may not be true on your device. Also sometimes it takes some time to calibrate. For this reason also the raw orientation data is shown. When you turn round once the first angle should increase from 0° to 360°. The two other angles are often much more reliable.

Beschreibung

Dies ist eine kombinierte Lineal- und Winkelmesser-App für Firefox OS.

Das Lineal verwendet sowohl cm als auch inch, tippe die Beschriftung an um die Seiten zu vertauschen. Der Winkelmesser misst den Winkel der langen Kante des Geräts (zumindest ist es bei Smartphones die lange Kante) gegenüber der Position, an der sie sich beim Zurücksetzen des Winkelmessers befand.

Beachte, dass der Winkelmesser auf genaue Daten vom Kompass angewiesen ist, was nicht auf allen Geräten der Fall ist. Auch dauert es manchmal eine gewisse Zeit bis er korrekt kalibriert ist. Daher werden auch die Rohdaten angezeigt. Wenn du dich einmal im Kreis drehst, sollte der erste Winkel von 0° bis 360° anwachsen. Die beiden anderen Winkel sind häufig wesentlich zuverlässiger.

Install / Installieren

The online version allows to quickly test the app in any browser. Please note that the app was developed for mobile phones with Firefox OS, so the online version might have bugs.

Die Online-Version erlaubt ein schnelles Testen der App in einem beliebigen Browser. Beachte, dass die App aber für Mobilgeräte mit Firefox OS entwickelt wurde, sodass die Online-Version Fehler besitzen kann.

Use online version / Online-Version verwenden

On devices with Firefox OS you can install the app and use it offline.

Auf Geräten mit Firefox OS lässt sich die App installieren und benötigt dann keine Online-Verbindung mehr.
